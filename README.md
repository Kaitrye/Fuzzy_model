# Нечёткая система управления температурой воды в душе

Проект реализует систему нечёткого вывода для управления комфортной температурой воды в душе.  
Система получает на вход отклонение текущей температуры от целевой и скорость изменения температуры, а на выходе определяет углы открытия горячего и холодного кранов.

## Описание

В работе рассматривается задача автоматического управления температурой воды в душе. Комфортной считается температура в диапазоне 36–40 °C, целевое значение — 38 °C.

Модель построена на основе алгоритма Мамдани. Для входных и выходных переменных заданы треугольные и трапециевидные функции принадлежности. База правил содержит 15 продукционных правил.

Дополнительно в модели поддерживается постоянный суммарный напор воды: после дефаззификации углы открытия кранов нормируются так, чтобы их сумма была равна 90°. Благодаря этому система управляет соотношением горячего и холодного потоков, а не общим расходом воды.

## Используемые технологии

- Python
- NumPy
- Matplotlib

## Результаты

В проекте строятся:

- графики функций принадлежности;
- поверхности нечёткого вывода для горячего и холодного кранов;
- графики динамического моделирования при разных начальных температурах воды.

Динамическая проверка показывает, что система постепенно приводит температуру воды к комфортной зоне и удерживает её около целевого значения.
